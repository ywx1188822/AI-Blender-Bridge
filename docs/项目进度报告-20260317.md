## 📁 新增文件

### video_gen.py (350 行)

**核心功能**:
- `AIBRIDGE_OT_GenerateWanVideo` 操作器
- Wan 2.2 工作流加载和更新
- ComfyUI 任务提交和轮询
- 视频文件下载和加载

**参数支持**:
| 参数 | 默认值 | 说明 |
|------|--------|------|
| image_path | - | 输入图像路径 |
| prompt | "smooth animation..." | 提示词 |
| negative_prompt | "ugly, deformed..." | 负向提示词 |
| steps | 30 | 采样步数 |
| cfg_scale | 7.0 | CFG 引导系数 |
| seed | -1 | 随机种子 |
| video_length | 48 | 视频帧数 |
| fps | 24 | 帧率 |

### wan22_video.json

**工作流节点**:
1. LoadImage - 加载输入图像
2. WanVideoModelLoader - 加载 Wan 视频模型
3. CLIPTextEncode - 正向提示词
4. CLIPTextEncode - 负向提示词
5. WanVideoSampler - 视频生成采样
6. SaveAnimatedWEBP - 保存 WEBP 动画
7. SaveAnimatedPNG - 保存 PNG 动画

**支持模型**:
- wan2.1-i2v-720p-14B (高画质)
- wan2.1-i2v-480p-14B (标准画质)
- smoothMix (动作流畅)

## 🐛 已知问题

### 问题 1: Wan 2.2 节点依赖

**症状**: WanVideoModelLoader 节点可能不存在

**解决**:
- 需要安装 Wan 2.2 自定义节点
- 或更新工作流使用标准节点

### 问题 2: 视频格式兼容性

**症状**: Blender 可能无法直接加载 WEBP/PNG 动画

**解决**:
- 使用 FFmpeg 转换为 MP4
- 或加载为图像序列
