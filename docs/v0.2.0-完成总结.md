## 🔧 技术实现

### Wan 2.2 视频生成流程

```
1. 加载输入图像 (LoadImage)
       ↓
2. 加载 Wan 模型 (WanVideoModelLoader)
       ↓
3. 编码提示词 (CLIPTextEncode x2)
       ↓
4. 视频采样 (WanVideoSampler)
       ↓
5. 保存动画 (SaveAnimatedWEBP/PNG)
       ↓
6. 下载并加载到 Blender
```

### 核心类

```python
class AIBRIDGE_OT_GenerateWanVideo(bpy.types.Operator):
    """使用 Wan 2.2 生成视频"""
    
    # 参数
    - image_path: 输入图像
    - prompt: 提示词
    - negative_prompt: 负向提示词
    - steps: 采样步数 (30)
    - cfg_scale: CFG 引导 (7.0)
    - seed: 随机种子
    - video_length: 视频帧数 (48)
    - fps: 帧率 (24)
    
    # 方法
    - execute(): 执行生成
    - load_wan_workflow(): 加载工作流
    - update_workflow(): 更新参数
    - submit_to_comfyui(): 提交任务
    - wait_for_completion(): 轮询状态
    - load_video_to_blender(): 加载视频
```
