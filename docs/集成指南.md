## 📋 集成方案

### 方案 A: 作为插件集成 (推荐)

将 AI Blender Bridge 作为 ComfyUI-BlenderAI-node 的插件模块。

### 方案 B: 独立安装

作为独立 Blender 插件安装，通过 ComfyUI API 通信。

---

## 🔧 方案 A: 插件集成步骤

### 步骤 1: Fork 项目

1. 访问 https://github.com/AIGODLIKE/ComfyUI-BlenderAI-node
2. 点击右上角 "Fork" 按钮
3. 创建你的 Fork 版本

### 步骤 2: 克隆项目

```bash
cd /path/to/your/projects
git clone https://github.com/YOUR_USERNAME/ComfyUI-BlenderAI-node.git
cd ComfyUI-BlenderAI-node
```

### 步骤 3: 复制插件文件

```bash
# 复制 AI Blender Bridge 插件到 SDNode/plugins 目录
cp -r /path/to/ai-blender-bridge/blender-plugin \
      /path/to/ComfyUI-BlenderAI-node/SDNode/plugins/ai_blender_bridge
```

### 步骤 4: 注册插件

编辑 `SDNode/plugins/__init__.py`，添加插件注册：

```python
# 在文件末尾添加
from .ai_blender_bridge import register as aibridge_register
from .ai_blender_bridge import unregister as aibridge_unregister

def register_all_plugins():
    # ... 现有代码 ...
    
    # 注册 AI Blender Bridge
    aibridge_register()

def unregister_all_plugins():
    # ... 现有代码 ...
    
    # 注销 AI Blender Bridge
    aibridge_unregister()
```

### 步骤 5: 添加依赖

编辑 `SDNode/plugins/ai_blender_bridge/__init__.py`，确保导入正确：

```python
bl_info = {
    "name": "AI Blender Bridge",
    "author": "AI Assistant",
    "version": (0, 1, 0),
    "blender": (3, 0, 0),
    "location": "View3D > Sidebar > AI Bridge",
    "description": "Short drama creation and character generation tools",
    "category": "AI",
}

import bpy
import sys
from pathlib import Path

# 添加插件目录到 Python 路径
plugin_dir = Path(__file__).parent
if str(plugin_dir) not in sys.path:
    sys.path.append(str(plugin_dir))

# 导入模块
from . import short_drama
from . import character_gen
from . import storyboard
from .ui import panel
from . import settings


def register():
    """注册插件"""
    print("🦞 AI Blender Bridge 插件正在加载...")
    
    # 注册设置
    settings.register()
    
    # 注册 UI 面板
    panel.register()
    
    # 注册业务模块
    short_drama.register()
    character_gen.register()
    storyboard.register()
    
    print("✅ AI Blender Bridge 插件加载完成")


def unregister():
    """注销插件"""
    print("🦞 AI Blender Bridge 插件正在卸载...")
    
    # 注销业务模块
    storyboard.unregister()
    character_gen.unregister()
    short_drama.unregister()
    
    # 注销 UI 面板
    panel.unregister()
    
    # 注销设置
    settings.unregister()
    
    print("✅ AI Blender Bridge 插件卸载完成")


if __name__ == "__main__":
    register()
```

### 步骤 6: 测试插件

1. 打开 Blender
2. 编辑 → 偏好设置 → 插件
3. 点击 "安装"
4. 导航到 `ComfyUI-BlenderAI-node/SDNode/plugins/ai_blender_bridge`
5. 选择并安装
6. 勾选启用
7. 在 3D Viewport 侧边栏查看 "AI Bridge" 面板

## 🔧 方案 B: 独立安装

### 步骤 1: 准备插件包

```bash
cd /path/to/ai-blender-bridge
zip -r ai_blender_bridge.zip blender-plugin/
```

### 步骤 2: 安装到 Blender

1. 打开 Blender
2. 编辑 → 偏好设置 → 插件
3. 点击 "安装"
4. 选择 `ai_blender_bridge.zip`
5. 勾选启用

### 步骤 3: 配置 ComfyUI 连接

1. 在 3D Viewport 侧边栏打开 "AI Bridge" 面板
2. 设置 ComfyUI 服务器地址：`192.168.3.86:8188`
3. 配置 LLM API Key

## 🐛 常见问题

### 问题 1: 模块导入错误

**症状**: `ModuleNotFoundError: No module named 'xxx'`

**解决**:
```python
# 在 __init__.py 中添加
import sys
from pathlib import Path
plugin_dir = Path(__file__).parent
if str(plugin_dir) not in sys.path:
    sys.path.append(str(plugin_dir))
```

### 问题 2: 面板不显示

**症状**: 侧边栏没有 "AI Bridge" 标签

**解决**:
1. 确认插件已启用
2. 重启 Blender
3. 检查 `bl_category` 设置

### 问题 3: ComfyUI 连接失败

**症状**: "Connection refused" 错误

**解决**:
1. 检查 ComfyUI 是否运行
2. 确认服务器地址正确
3. 检查防火墙设置
